Monedero - Simple personal wallet

Context

Web app to track balances and transactions for personal use. Focus on clarity and safety.

Problem

Users struggle to confirm balance and find past transactions. We need a simple mental model and safe actions (undo, confirmations).

Process

  • Defined IA: Accounts, Transactions, Categories, Settings.
  • Flow for add income/expense, filter/search, export.
  • Wireframes -> UI with dense table and mobile cards.
  • Empty/edge states, error handling, destructive actions patterns.

Solution

Overview dashboard with balance cards; transactions table with filters; add flow with category pills; edit/undo snackbar.
